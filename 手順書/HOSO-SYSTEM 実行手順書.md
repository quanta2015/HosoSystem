## 2.1. 営業所管理

営業所リスト：
![dep list](./img/dep/list.png)
| id  |     説明      |
| :-- | :---------: |
| ①   | 営業所を追加（詳しくは[2.1.1](#211)へ）|
| ②   | テーブルヘッダー (詳しくは[2.1.2](#212)へ)|
| ③   | 営業所の詳細を表示（詳しくは[2.1.1](#211)へ）  |
| ④   | 営業所の情報を編集 （詳しくは[2.1.1](#211)へ） |
| ⑤   | 営業所を削除（詳しくは[2.1.3](#213)へ） |

<a id="211"></a>
### 2.1.1. 営業所を追加（①）、編集（④）、詳細の表示（③）

この三つのボタン全部同じUIが表示される。\
追加する場合は空っぽUI。\
編集する場合は指定されたレコードのデーターが入力フィールドに入る。\
詳細を表示する場合は編集と同じで、入力フィールドが編集できない状態になる。\

追加ボタンを押すと以下のUIが表示される
![dep create](./img/dep/add.png)
編集ボタンを押すと以下のUIが表示される
![dep edit](./img/dep/edit.png)
詳細ボタンを押すと以下のUIが表示される
![dep edit](./img/dep/detail.png)

| id  |     説明      |
| :-- | :---------: |
| ①   | 営業所の名前を入力 |
| ②   | 営業所のアドレスを入力 |
| ③   | 営業所の連絡先を入力   |
| ④   | 他の情報を追加（押すとその他に1行追加する）   |
| ⑤   | 情報名|
| ⑥   | 情報内容   |
| ⑦   | 行を削除する   |
| ⑧   | 保存せずにUIを消す   |
| ⑨   | 保存してUIを消す |


<a id="212"></a>
### 2.1.2 ②：テーブルヘッダー

カラム名を表示する。\
虫眼鏡アイコン![search icon](./img/search-icon.png)がついているカラムをクリックしたら検索できる。\
![dep search2](./img/dep/search2.png)
![dep search](./img/dep/search.png)

<a id="213"></a>
### 2.1.3 ⑤:営業所を削除

このボタンを押すと確認メッセージが表示され、はいを押すと指定された記録が削除される。\
確認メッセージの画像：\
![dep remove warning](./img/dep/remove.png)
